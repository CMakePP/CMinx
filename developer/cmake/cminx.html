cminx.cmake¶

cminx.cmake.cminx_gen_rst(_cgd_dir _cgd_output)¶

Generate documentation RST from source CMake files.

Parameters

  • dir (dir) – Directory to search for source files. Can also be a single file.

  • output (dir) – Directory to store output

  • *args – Optional arguments below. Pass to the function after all other parameters, order-sensitive.

Optional Arguments

  • Prefix (string) – A prefix to be passed to CMinx. All files will have the prefix prepended to their RST titles, and root-level files will have their titles replaced by the prefix in recursive mode.

cminx.cmake.cminx_add_docs_target(_cad_doc_dir _cad_output_dir _cad_sphinx_target)¶

Add a custom target to the makefile called “docs” that calls Sphinx. Sphinx is automatically installed in the virtual environment and the target automatically uses that installed instance.

Parameters

  • _cad_doc_dir (dir) – Root directory of the sphinx docs where the makefile is located.

  • _cad_output_dir (dir) – Directory where built docs will be placed

  • _cad_sphinx_target (desc) – Requested build target for Sphinx docs. Most common is “html”.
